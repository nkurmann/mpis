== Description ==

This plugin takes remote page, examines its content and lists all found images for your choice. 
Then you are able to select which images you want to import, add some fancy options such as re-size, rename, add your personal titles and more and then in one click to import all images into your wordpress media gallery and automatically create post with imported images in it (gallery or embedded with titles). 
Very easy and handy tool if you are running blog and often need to import someone else images. 
Moreover link to source will be added too if needed. 
Post may be fully reviewed and edited before published.

== Installation ==

1. Upload `image-leech.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To import images go to Media -> Import Gallery
